coop-coffees-members
====================

Plone + Python
--------------

::

    git clone git@github.com:ACLARKNET/coop-coffees-members.git
    sudo mv coop-coffees-members /srv/
    cd /srv/coop-coffees-members
    tar zxvf Python-2.4.6.tgz
    cd Python-2.4.6
    ./configure
    make
    sudo make install
